PROYECTO MARVEL

Acerca del proyecto

Esta aplicación fue creada con el fin de mostrar los personajes de Marvel, sus comics, eventos, historias y series. Para ello, se utilizó la API pública de Marvel. Además, se creó una e-commerce de los productos de la marca, el cual utiliza la pasarela de pagos Stripe. También a creación de una cuenta de usuario, y loguearse con ella. Además, de poder cambiar de idioma entre español e inglés; y el tema entre claro y oscuro. Todo lo anterior se realiza utilizando React, React Router, Firebase, Stripe, Bootstrap, entre otras tecnologías.

Desafíos

Escoger un tema que nos guste y que sea atractivo para los usuarios, en este caso, los personajes de Marvel, sus comics, eventos, historias y series; para la creación de la pagina web.

Decidir las tecnologias a utilizar en el proyecto y como implementarlas. Por ejemplo, el uso de Firebase para la autenticación de usuarios, el uso de Stripe para la pasarela de pagos, el uso de React Router para el enrutamiento de la aplicación, entre otros.

Crear una aplicación que sea responsive, es decir, que se adapte a cualquier dispositivo.

Leer documentación técnica de las herramientas a utilizar,por ejemplo de la API de Marvel, para poder implementarlas correctamente.

Adecuado uso y manejo de la herramienta de git y github para el control de versiones

Trabajo en equipo

Manejo de formato de fechas y monedas

Aprendizajes

Conceptos vistos en el curso de React, como por ejemplo: componentes funcionales, props, state, ciclo de vida de los componentes, hooks, enrutamiento, entre otros.

Consumo de la API pública de Marvel, usando llaves públicas y privadas.

Uso de Firebase para la autenticación de usuarios, creación de usuarios, login, logout, entre otros.

Uso de Stripe para la pasarela de pagos, creación de productos, carrito de compras, checkout, entre otros.

Uso de Bootstrap para el diseño de la página web, creación de componentes, entre otros.

Uso de React Context para el manejo de estados globales, creación de contextos, uso de useContext, entre otros.
